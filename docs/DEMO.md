# 75-second Demo Script

1. **0–10s — Positioning**: “Stripe moves money; Clearance decides whether an AI agent is allowed to.”
2. **10–28s — Approved request**: Run the 1 HBAR scan. Point out seven policy verdicts and that signing authority is absent from Gemini. Click approval, show the real HashScan receipt, then open the paid GitHub evidence.
3. **28–43s — Blocked request**: Run the 25 HBAR unknown-vendor scenario. Emphasize “Blocked before transaction creation” and show amount, budget, and counterparty failures.
4. **43–57s — Contextual escalation**: Run the production audit. Show the typed `APPROVE 4 HBAR` requirement and explain that risk changes the authorization ceremony.
5. **57–68s — Replay and audit**: Replay the lifecycle without resubmitting. Show the HCS/HashScan evidence and distinction between audit and payment transactions.
6. **68–75s — Close**: “Clearance makes autonomous commerce governable. Hedera makes the result verifiable.”

Never record placeholder transaction IDs. If the network is unavailable, show the policy flows and state the limitation plainly.
